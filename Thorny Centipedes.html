Thorny Centipedes




Stats:


HP: 600

MP: 0

PHY STR: 18

PHY DEF: 15

MG STR: 8

MG DEF: 12













Thorny Centipedes Skills:

  • Venomous Bite (Offensive) (Level 0): Deals 250 DMG to a single target with a 30% chance to apply a Low-Tier Poisoned status effect for 5 turns.
  • Thorny Armor (Support) (Level 5): Increases the user's PHY DEF by 15% for 3 turns. Enemies attacking the user during the duration of this skill will have a 10% chance to be afflicted by a Low-Tier Poisoned status effect for 3 turns. (Can only be used once per battle)
  • Toxic Spit (Offensive) (Level 15): Deals 450 DMG to a single target and applies a Mid-Tier Poisoned status effect for 5 turns.
  • Entangling Thorns (Support) (Level 30): Summons a tangle of thorns, dealing 250 DMG to all enemies present for 3 turns and reducing their PHY DEF by 10% (Can only be used three times per battle).
  • Venomous Surge (Offensive) (Level 40): Deals 300 DMG to all enemies in a cone in front of the user and applies a Mid-Tier Poisoned status effect for 5 turns. The Poisoned tier will increase to High-Tier if the target is already afflicted by the Poisoned status effect.
  • Toxic Regeneration (Support) (Level 55): Removes all negative status effects from the user and heals them for 5% of their max HP. Additionally, the user gains a 20% damage reduction for 3 turns. (Can only be used twice per battle).
  • Lethal Venom (Offensive) (Level 70): Deals 700 DMG to a single target and applies a High-Tier Poisoned status effect for 5 turns.
  • Hive's Venom (Support) (Level 80): The user gains a 15% damage increase and a 10% increase in PHY DEF. Additionally, all their attacks have a 30% chance to apply a High-Tier Poisoned status effect for 5 turns. (Can only be used three times per battle).
  • Hive's Wrath (Offensive) (Level 90): Deals 900 DMG to all enemies in a wide area around the user and applies a High-Tier Poisoned status effect for 7 turns. The Poisoned duration increases by 3 if the target is already afflicted by the Poisoned status effect.
  • Venomous Onslaught (Offensive) (Level 100): Deals 1200 DMG to a single target and applies a High-Tier Poisoned status effect for 10 turns. If the target survives the attack, they are stunned for one turn and the user gains a 30% PHY ATK increase. (Can only be used once per battle).
