Webspinner Spiders




Stats:


HP: 530

MP: 0

PHY STR: 15

PHY DEF: 12

MG STR: 20

MG DEF: 15













Webspinner Spiders Skills:

  • Spider Bite (Offensive) (Level 0): Deals 200 DMG to a single target with a 15% chance to apply the Black Heart status effect.
  • Web Snare (Support) (Level 5): The user creates a web trap in a targeted area, dealing 200 DMG to all enemies for 3 turns. Bound enemies have a 20% chance to be afflicted by the Black Heart status effect for 3 turns. (Can only be used once per battle).
  • Web Shield (Support) (Level 10): The user weaves a protective web around themselves, gaining a 15% damage reduction for 3 turns. Enemies attacking the user during the duration of this skill will have a 10% chance to be ensnared by webs, preventing their next action. (Can only be used twice per battle).
  • Entangle (Offensive) (Level 20): The user entangles a single target with webs, dealing 300 DMG and binding them for 2 turns. If the target is already bound by an attack, they take an additional 150 DMG and have a 30% chance to be afflicted by the Black Heart status effect for 5 turns (Can only be used twice per battle).
  • Spider's Leap (Support) (Level 30): The user leaps to a targeted location, dealing 200 DMG to enemies in the area and gaining a 20% PHY ATK increase for 2 turns. Enemies hit by the leap have a 25% chance to be afflicted by the Black Heart status effect for 1 turn. (Can only be used three times per battle)
  • Web of Shadows (Support) (Level 45): The user creates a veil of shadows, becoming invisible to enemies for 3 turns. While invisible, the user gains a 30% damage reduction and a 15% PHY ATK increase. Enemies that attack the user during this time have a 20% chance to be ensnared by webs, preventing their next action. (Can only be used once per battle).
  • Venomous Silk (Support) (Level 60): The user coats their weapons in venomous silk, increasing their next attack's damage by 50%. If the attack hits, the target is afflicted by the Black Heart status effect for 5 turns. (Can only be used once per battle).
  • Spider's Embrace (Offensive) (Level 75): The user grapples a single target and deals 500 DMG, while healing the user for 50% of that targetâ€™s max HP. The target is then bound for 2 turns and has a 50% chance to be afflicted by the Black Heart status effect for 7 turns (Can only be used twice per battle).
  • Web of Destiny (Support) (Level 90): The user weaves a powerful magic web around a targeted ally, increasing their PHY ATK by 30% and granting them immunity to the Instant Death status effect for 3 turns. Enemies attacking the ally have a 30% chance to be ensnared by webs, preventing their next action and a 25% chance to be afflicted by the Black Heart status effect for 3 turns. (Can only be used on an ally) (Can only be used twice per battle).
  • Spider's Web (Ultimate) (Level 100): The user spins an enormous web, dealing 1500 DMG to all enemies present and binding them for 3 turns. Bound enemies have a 50% chance to be afflicted by the Black Heart status effect for 7 turns. The user gains a 50% PHY ATK increase for the duration of the skill. (Can only be used once per battle).
